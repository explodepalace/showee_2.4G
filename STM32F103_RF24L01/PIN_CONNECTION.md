# STM32F103C8T6 + RF24L01 引脚连接图

## 📌 开发板IO口现状

**已修改为适配STM32F103C8T6（无PG口）**

### 🔌 RF24L01 8引脚连接

```
     RF24L01
    ┌─────────┐
  1 │ GND     │ 8 VCC (3.3V)
  2 │ MOSI    │ 7 CE
  3 │ MISO    │ 6 CSN
  4 │ SCK     │ 5 IRQ
    └─────────┘
```

## 📍 详细连接表

| RF24L01引脚 | 引脚号 | 功能 | STM32连接 | 说明 |
|-----------|--------|------|----------|------|
| **GND** | 1 | 地线 | GND | 电源地 |
| **MOSI** | 2 | SPI数据输入 | **PB15** | SPI主出从入 |
| **MISO** | 3 | SPI数据输出 | **PB14** | SPI主入从出 |
| **SCK** | 4 | SPI时钟 | **PB13** | SPI时钟信号 |
| **IRQ** | 5 | 中断信号 | **PA2** | 低电平有效 |
| **CSN** | 6 | 片选信号 | **PA3** | 低电平有效 |
| **CE** | 7 | 芯片使能 | **PA4** | 高电平使能 |
| **VCC** | 8 | 电源 | **3.3V** | 供电 |

## 📊 STM32F103 全部引脚分配

| 功能 | 端口 | 引脚 | 说明 |
|------|------|------|------|
| **UART TX** | GPIOA | PA9 | 串口发送 |
| **UART RX** | GPIOA | PA10 | 串口接收 |
| **RF24 IRQ** | GPIOA | PA2 | RF24L01中断 |
| **RF24 CSN** | GPIOA | PA3 | RF24L01片选 |
| **RF24 CE** | GPIOA | PA4 | RF24L01使能 |
| **SPI CLK** | GPIOB | PB13 | SPI时钟 |
| **SPI MISO** | GPIOB | PB14 | SPI数据输入 |
| **SPI MOSI** | GPIOB | PB15 | SPI数据输出 |
| **LED** | GPIOC | PC13 | 状态指示灯 |

## ⚡ 关键注意事项

1. **电源**：RF24L01必须用**3.3V**供电，不能用5V
2. **去耦电容**：在VCC和GND之间接**10µF + 100nF**电容，稳定电源
3. **SPI接口**：使用硬件SPI2
   - MOSI (引脚2) → PB15
   - MISO (引脚3) → PB14
   - SCK (引脚4) → PB13
4. **控制引脚**（软件控制）：
   - CE (引脚7) → PA4 (高电平激活收发)
   - CSN (引脚6) → PA3 (低电平选中芯片)
   - IRQ (引脚5) → PA2 (中断信号，低电平有效)

## 🎯 简化版接线清单

```
RF24L01 → STM32F103C8T6
─────────────────────────
1 (GND)  → GND
2 (MOSI) → PB15
3 (MISO) → PB14
4 (SCK)  → PB13
5 (IRQ)  → PA2
6 (CSN)  → PA3
7 (CE)   → PA4
8 (VCC)  → 3.3V (+ 去耦电容)
```

## 📌 LED指示灯

- **PC13** → 单个LED
  - 快速闪烁（200ms周期）：TX模式/发送中
  - 慢速闪烁（500ms周期）：RX模式/接收中
  - 熄灭：待机

## 🔄 数据流向

```
【发送模式 TX_MODE_2】
电脑 ──(串口PA9/PA10)──→ STM32 ──(SPI2)──→ RF24L01 ──(无线)──→ 对方

【接收模式 RX】
对方 ──(无线)──→ RF24L01 ──(SPI2)──→ STM32 ──(串口PA9/PA10)──→ 电脑
```
